h1. Welcome to UML Embedded C Generation from Eclipse UML Generators.

This project provides a UML to Embedded C generator created with Acceleo to generate Embedded C code from UML models.

The master branch supports UML2 V4.x

These plugins are released under the EPL Open-Source License.

Acceleo is an implementation of the OMG specification MOF Model-To-Text Transformation Language (MTL)
Visit http://www.eclipse.org/acceleo

UML is the specification from OMG. The current version is 2.x.
Visit http://www.omg.org/spec/UML/
